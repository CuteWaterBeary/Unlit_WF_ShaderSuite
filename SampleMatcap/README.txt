================================================================
SampleMatcap
================================================================

このフォルダには MatcapStudio にて撮影した matcap 用テクスチャが入っています。
ご自由にお使いください。

matcap1.png
    Standard Shader 製、非金属の柔らかい反射と影の matcap です。

matcap2.png
    Standard Shader 製、matcap1 の色合いを黄色に近づけ、肌色に馴染むように作成した matcap です。

matcap3.png
    Standard Shader 製、金属系の反射と影の matcap です。

matcap4.png
    トゥーンカラーの matcap です。作成には UnityChanToonShader 2.0 を使用させていただきました。
